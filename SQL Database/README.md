## Database Setup

### If you use XAMPP (phpMyAdmin):
1. Open phpMyAdmin → create database → name it school_project
2. Click Import → select migrations/001_create_users_table.sql → Go
3. Repeat for each migration file in order

### If you use MySQL Workbench:
1. Create schema named school_project
2. File → Open SQL Script → select migration file → Run
3. Repeat for each migration file in order
